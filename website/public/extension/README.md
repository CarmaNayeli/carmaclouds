# OwlCloud - DiceCloud Integration for Owlbear Rodeo

Bring your DiceCloud V2 characters into Owlbear Rodeo with seamless integration.

## Installation

1. **Install the OwlCloud browser extension** from the [Chrome Web Store](#) or [Firefox Add-ons](#)

2. **Add the Owlbear extension** to your room:
   - Open [Owlbear Rodeo](https://www.owlbear.rodeo) and join/create a room
   - Click the Extensions menu (puzzle piece icon)
   - Click "Add Extension"
   - Paste this URL:
   ```
   https://carmaclouds.vercel.app/extension/manifest.json
   ```

3. **Select your character** in the OwlCloud browser extension popup

4. **Open the extension** in Owlbear to view your character sheet!

## Features

- View DiceCloud character sheets in Owlbear Rodeo
- Real-time HP tracking
- Roll dice from your character sheet
- Sync ability scores, stats, and resources
- Multiple character support

## How It Works

The browser extension syncs your DiceCloud characters and makes them available to the Owlbear extension through a secure connection. Your character data is cached locally for fast access.
